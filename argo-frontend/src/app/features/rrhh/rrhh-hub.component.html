Recursos humanos

Siga el flujo del mes en orden. Para abrir cada módulo use el menú RRHH del panel izquierdo.

Flujo recomendado

1 Catálogos Cargos, EPS, AFP… → 2 Empleados Ficha y seguridad social → 3 Contratos Salario y fechas → 4 Novedades IGE, licencias, PILA → 5 Nómina Generar y liquidar → 6 Caja Pagar neto

Liquidación del período

  1. Crear período en Liquidación
  2. Registrar novedades PILA si aplica (incapacidades, licencias)
  3. Generar novedades → Liquidar
  4. Descargar planilla PILA (.txt / .csv)
  5. Cerrar período y pagar el neto en Flujo de caja

Configuración previa (menú RRHH)

  • Parámetros legales — SMLMV, UVT y porcentajes del año vigente
  • Empresa (NIT) — datos del aportante para PILA y comprobantes
  • Cargos y seguridad social — EPS, AFP, ARL y cajas de compensación
  • Empleados con cargo Cajero o Instructor: usuario automático con el número de documento
Importar administradoras: si tiene el Excel del Ministerio, en el servidor ejecute pnpm run import:administradoras (carpeta excel/) para cargar códigos EPS, AFP, ARL y CCF.
